# قانونك — Qanunak

منصة تشرح القوانين البريطانية بالعربية بطريقة بسيطة، مع مصدر حكومي رسمي لكل موضوع.

## التشغيل
```bash
cp .env.example .env   # ثم غيّر كلمة المرور والسر
npm install
npm run dev
```
- الموقع: http://localhost:3000
- لوحة التحكم: http://localhost:3000/admin

راجع CLAUDE.md للتفاصيل المعمارية وقواعد المشروع.
